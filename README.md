End-to-end CI/CD and GitOps platform built with Terraform, Jenkins, Argo CD, Helm, Kubernetes and AWS EKS.

Overview
This project demonstrates a complete CI/CD and GitOps workflow for deploying applications on Amazon EKS. It provisions the infrastructure using Terraform, automates builds and quality checks with Jenkins, and manages Kubernetes deployments using Argo CD and Helm.

Technologies Used

- Terraform
- AWS EKS
- Kubernetes
- Jenkins
- Maven
- SonarQube
- Docker
- Helm
- Argo CD
